# MyIdentity
Progetto realizzato nell'ambito del corso di Ingegneria del Software, dedicato alla definizione delle specifiche per lo sviluppo di **MyIdentity**. La repository include il documento di *Software Requirements Specification (SRS)*, contenente i requisiti funzionali e non funzionali del sistema, i casi d'uso, gli scenari operativi e la relativa modellazione mediante diagrammi UML e BPMN, nonché i design pattern individuati durante la fase di progettazione. Sulla base delle specifiche definite, è stata sviluppata un'implementazione parziale del progetto in linguaggio Java, corredata da una *suite di test* realizzata con il framework JUnit.
